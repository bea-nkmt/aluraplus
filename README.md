# aluraplus

Alura Plus - Landing Page
Esta é uma página de vendas (Landing Page) desenvolvida para o projeto Alura Plus. O objetivo do projeto foi aplicar conceitos modernos de Front-end, focando em posicionamento de elementos, responsividade básica e organização de código.

🚀 Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização e layout.

Google Fonts: Integração da fonte 'Inter' para uma tipografia moderna.

🛠️ Conceitos Aplicados
Durante o desenvolvimento, foram colocados em prática diversos conceitos fundamentais:

CSS Grid: Utilizado no container principal para criar a divisão de colunas (50% 50%).

Flexbox: Aplicado no rodapé e na seção de dispositivos para alinhamento e distribuição dos itens.

Variáveis CSS: Uso de :root para gerenciar a paleta de cores (Branco, Cinza, Azul e Cor de Fundo) de forma centralizada.

Pseudo-classes: Implementação de efeitos de :hover e :active nos botões e links para melhorar a experiência do usuário.

Box Model: Reset global utilizando box-sizing: border-box para um controle mais preciso de margens e preenchimentos.

🎨 Layout
O layout segue uma estética de serviços de streaming, com:

Seção Principal: Chamada para assinatura com imagem de destaque.

Seções Informativas: Alternância entre imagens e textos explicativos sobre os benefícios do plano.

Dispositivos: Lista visual dos aparelhos compatíveis.

Rodapé: Links institucionais e informações legais.
